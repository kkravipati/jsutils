Replace (part of) an array with items from another array.

parameter type description
arr Array The array to manipulate.
startIndex Number Start index of the part of the array to be replaced, it should be an integer >= 0.

indexOf (array, searchElement, fromIndex, isEqualFunc?)

Search for an element in an Array.

parameter type description
array Array The array to be searched.
searchElement any The element to search for.
fromIndex Number = 0 The index to start searching from. It should be an integer >= 0.
isEqualFunc Function? A custom equality function to use for the search.

Returns

int : Element search Index. If element is not found, it returns -1.

Examples

ArrayUtils.indexOf(['a', 'ab', 'abc'], 'ab');
// => 1

ArrayUtils.indexOf([[1, 2], null, [1, 3], undefined, 'a'], [1, 3], 0,
 (a, b) => ((a === b) || (a && b && a.toString() === b.toString())));
// => 2

isEqual (value, other)

Performs a deep comparison between two values to determine if they are equivalent (uses Lodash isEqual method).

Note: This method supports comparing arrays, array buffers, booleans, date objects, error objects, maps, numbers, Object objects, regexes, sets, strings, symbols, and typed arrays. Object objects are compared by their own, not inherited, enumerable properties. Functions and DOM nodes are not supported.

parameter type description
value any The value to compare.
other any The other value to compare.

Returns

Boolean : true if the values are equivalent, else false .

Examples

var object = { 'user': 'fred' };
var other = { 'user': 'fred' };

LangUtils.isEqual(object, other);
// => true

object === other;
// => false

merge (toDict, fromDict, recursive, notOverride, ignoreNull, extendObjectArray)

Merge one object into another.

parameter type description
toDict Object Object on merge operation is going to perform.
fromDict Object Object to be merged.
recursive Boolean = false Recursive merge or not.
notOverride Boolean = false Overide or not. by default override is true.
ignoreNull Boolean = false Consider null values & keys or not. By default null values will be merged.
extendObjectArray Boolean = false Override / extend arrays.by defult override arrys is on.

Examples

let toDict = {a: '1', b: {c: '1', e: null}, f: [{g: '1'}] };
let fromDict = {b: {c: '2', d: '2', e: '2'}, f: [{g: '2'}] };
ObjectUtils.merge(toDict, fromDict);
// => toDict is updated as {a: '1', b: {c: '2', d: '2', e: '2'}, f: [{g: '2'}]}

let toDict = {a: '1', b: {c: '1', e: null}, f: [{g: '1'}] };
let fromDict = {b: {c: '2', d: '2', e: '2'}, f: [{g: '2'}] };
ObjectUtils.merge(toDict, fromDict, true, true, true, true);
// => toDict is updated as {a: '1', b: {c: '1', d: '2', e: '2'}, f: [{g: '1'}, {g: '2'}]}

let toDict = {a: '1', b: {c: '1', e: null}, f: [{g: '1'}] };
let fromDict = {b: {c: '2', d: '2', e: '2'}, f: [{g: '2'}] };
ObjectUtils.merge(toDict, fromDict, true, true, false, true);
// => toDict is updated as {a: '1', b: {c: '1', d: '2', e: null}, f: [{g: '1'}, {g: '2'}]}
